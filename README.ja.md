# kids

子ども霞が関見学デー2016/2017の検索アプリ

## デモ
https://fukuno.jig.jp/app/kids/index.html
https://fukuno.jig.jp/app/kids/index-2016.html

## 機能
- 子ども霞が関見学デーの一覧表示
- カテゴリ（学ぶ、ふれる、体を動かす等）と対象年齢（就学前、小学生等）による絞り込み検索
- キーワード検索

## データ・API
オープンデータ「平成28年度子ども霞が関見学デー プログラム」および「平成29年度こども霘が関見学デー プログラムオープンデータ」を利用しています。
