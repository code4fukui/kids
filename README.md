# kids

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A search application for the "Children's Kasumigaseki Viewing Day" program.

## Features
- Search and filter events by category (learn, touch/feel, move body, play, see, create, harvest/collect) and target audience (pre-schoolers, lower/upper elementary, middle school, high school)
- Search by keyword

## Data / API
The application uses open data from the Google Sheets "[平成29年度こども霞が関見学デー プログラムオープンデータ](https://docs.google.com/spreadsheets/d/1FEhoCeSrnW-D8YLSfy3wMDQcSABQf1qfZtz4jpwZ5sg/edit?usp=sharing)".
